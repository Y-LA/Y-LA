## 🚀 Featured Projects

### 🌿 AI Plant Disease Diagnosis System

An end-to-end **AI + IoT graduation project** that helps users diagnose plant diseases from leaf images and monitor environmental conditions.

The system combines a fine-tuned ResNet50 computer vision model, FastAPI backend, Flutter mobile app, Firebase integration, and ESP32-S3 sensors for temperature, humidity, and light monitoring.

**Key Features**
- Plant disease classification using ResNet50 transfer learning
- Custom `Not-Plant` class to reduce false predictions
- FastAPI backend for AI predictions and sensor-data ingestion
- ESP32-S3 integration with DHT22 and LDR sensors
- Flutter mobile app with Arabic/English support
- Firebase authentication and diagnosis history

**Tech Stack:** Python, PyTorch, ResNet50, FastAPI, Flutter, Firebase, ESP32-S3, DHT22, LDR  
**Repository:** [plant-disease-ai](https://github.com/Y-LA/plant-disease-ai)
